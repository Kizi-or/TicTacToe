# TicTacToe

Simple game, created for training c++ and SFML library.
Functionalities are:
• Game end, when three X'es are in line or three circles
• After finish you cannot put your move on the board
• It is possible to fast refresh game by clicking "M"
• The lower part of the window navigates the game state
